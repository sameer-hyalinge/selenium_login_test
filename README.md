# SeleniumPractice 🧪🚀

A Java-based automation framework using **Selenium WebDriver**, **TestNG**, and **Maven**, enhanced with **Allure Reporting** and **GitHub Actions CI** for continuous testing.

---

## 🔧 Tech Stack

- 💻 **Java 21**
- 🌐 **Selenium WebDriver 4.20**
- 🧪 **TestNG 7.9**
- 📦 **Maven 3.9.10**
- 📊 **Allure Reports**
- ⚙️ **GitHub Actions CI/CD**
